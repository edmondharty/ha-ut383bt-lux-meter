# Uni-T (Uni Trend) Mini Sound Level Meter
This is a HACS component for the [Uni-T Mini Sound Level Meter UT353BT](https://meters.uni-trend.com/product/ut353-ut353bt/).

## Features

- Sensor: sound level in [dBA](https://www.nexflow.com/blog/what-does-dba-mean/).
- Binary Sensor: Battery Low (True/False)
- Text Sensor: MIN/MAX/Normal
- Text Sensor: Fast/Slow
- Set polling interval for reading above data
- Control: MIN/MAX, FAST/SLOW, Hold
